이미 1회차 가상환경이 있어요
기존 환경에 2회차 패키지만 추가 설치
기존 가상환경 활성화
Windows .venv\Scripts\activate
Mac source .venv/bin/activate

터미널 앞에 (.venv) 가 표시되면 활성화된 상태입니다.

2회차 추가 패키지 설치
pip install sentence-transformers lark langgraph numpy python-dotenv

1회차에 설치한 langchain, chromadb, streamlit 등은 이미 있으므로 추가분만 설치합니다.

⚠️
sentence-transformers는 PyTorch를 포함하고 있어서 설치에 3~5분 이상 걸릴 수 있습니다. 멈춘 것처럼 보여도 기다려주세요!
설치 확인
python -c "import sentence_transformers; import lark; import langgraph; print('OK!')"
✔️ OK!가 출력되면 성공! Section 4(설치 확인)로 건너뛰세요.

신규 설치 - 가상환경 생성부터 패키지 설치까지
Substep 1: 가상환경 생성
Windows python -m venv .venv
Mac python3 -m venv .venv
Substep 2: 가상환경 활성화
Windows .venv\Scripts\activate
Mac source .venv/bin/activate

터미널 앞에 (.venv) 가 표시되면 활성화 성공!

Substep 3: pip 업그레이드
pip install --upgrade pip
Substep 4: 패키지 설치

Option A: requirements.txt로 한번에 (권장)

권장 pip install -r requirements.txt

프로젝트 폴더에 requirements.txt 파일이 있어야 합니다.

Option B: 직접 설치

requirements.txt가 없거나 수동으로 설치하고 싶을 때:

pip install langchain langchain-community langchain-ollama chromadb streamlit sentence-transformers lark langgraph numpy python-dotenv
⚠️
sentence-transformers는 PyTorch를 포함하고 있어서 설치에 3~5분 이상 걸릴 수 있습니다. 멈춘 것처럼 보여도 기다려주세요!

문제 해결 (Troubleshooting)
설치 중 문제가 생겼을 때
설치가 멈추면

의존성 해결이 오래 걸리는 경우, 레거시 리졸버를 사용해보세요:

pip install -r requirements.txt --use-deprecated=legacy-resolver
가상환경이 꼬이면

기존 가상환경을 삭제하고 처음부터 다시 만드는 게 가장 깔끔합니다:

Windows rmdir /s /q .venv python -m venv .venv .venv\Scripts\activate pip install -r requirements.txt
Mac rm -rf .venv python3 -m venv .venv source .venv/bin/activate pip install -r requirements.txt
